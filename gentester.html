Gen examples

These are some gen examples that can be accepted into the textfield or uploaded as a .txt file

  • 1. The text between the "checklist = {}" in the .py or .txt gen files:

    checklist = {

         'example': 'here',
         'example': 'here',
         'example': 'here',

    }

  • 2. The entire "checklist = {}" py variable in the .py or .txt gen files:

    checklist = {

         'example': 'here',
         'example': 'here',
         'example': 'here',

    }

  • 3. The entire .py or .txt gen files:

    Upload the entire .txt/.py file, or paste the contents of the entire .txt/.py file directly into the textfield!

Note: The code works based on the "{}" surrounding the questions. So either submit exactly what is between those curly braces, or include the curly braces at the begining and end of the questions. i.e. If you decide to upload the entire gen file or partial file for whatever reason, the code will grab only the questions between those two curly braces. If the tool is not working with full files, use the #1 way of inserting gen.
