JS Create Boostrap Alert utility

Dynamic alert creation with custom configuration

  • type <string>:

    The type of alert to be displayed as defined by bootstrap design. w3Schools BS color guide.

  • location <string>:

    The fixed location for the alert to be displayed. (top/bottom)

  • duration <int>:

    Time interval (ms) for the alert to display before fading out.

    If the duration is 0 or less, the alert will persist until it is manually dismissed.

  • text <string>:

    The text to be displayed inside the alert box.
